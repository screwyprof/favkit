# favkit

A Rust library and CLI tool for managing macOS Finder sidebar items.

## Features

- List favorite items in the Finder sidebar
- (More features coming soon)

## Installation

```bash
cargo install favkit
```

## Usage

```bash
# List all favorite items
favkit list
```

## Development

```bash
# Build the project
cargo build

# Run tests
cargo test
```
